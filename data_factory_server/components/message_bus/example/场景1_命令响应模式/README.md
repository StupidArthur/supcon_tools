# 场景1：命令-响应模式（同步调用）

## 场景说明

演示如何使用消息中间件进行同步的服务调用。客户端发送命令后，等待服务端处理并返回响应。

## 运行方式

### 方式1：分别运行服务端和客户端

1. 启动服务端：
```bash
python 场景1_命令响应模式/service.py
```

2. 在另一个终端启动客户端：
```bash
python 场景1_命令响应模式/client.py
```

### 方式2：运行完整示例（包含服务端和客户端）

```bash
python 场景1_命令响应模式/完整示例.py
```

## 功能说明

- **服务端**：提供计算服务，接收两个数字，返回它们的和
- **客户端**：发送计算请求，等待并显示结果

## 预期输出

服务端输出：
```
服务端已启动，等待命令...
收到命令: calculate, payload: {'a': 10, 'b': 20}
计算结果: 30
```

客户端输出：
```
发送计算请求: a=10, b=20
收到响应: {'result': 30}
```
