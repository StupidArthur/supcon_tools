# 场景5：综合场景（多个服务交互）

## 场景说明

演示一个完整的微服务架构场景，包含多个服务之间的交互：
- 配置服务：管理配置
- 数据服务：处理数据
- Web服务：作为客户端调用其他服务

## 运行方式

```bash
python 场景5_综合场景/完整示例.py
```

## 功能说明

- **配置服务**：提供配置加载和更新功能，发布配置更新事件
- **数据服务**：提供数据处理功能，订阅配置更新事件
- **Web服务**：作为客户端，调用配置服务和数据服务

## 预期输出

```
[配置服务] 服务已注册: config_service
[配置服务] 服务端已启动
[数据服务] 服务已注册: data_service
[数据服务] 已订阅事件: config_updated
[数据服务] 服务端已启动
[Web服务] 发现配置服务
[Web服务] 加载配置: /path/to/config.yaml
[Web服务] 发现数据服务
[Web服务] 处理数据: test_data
[配置服务] 发布配置更新事件
[数据服务] 收到配置更新事件，执行配置重载
```
