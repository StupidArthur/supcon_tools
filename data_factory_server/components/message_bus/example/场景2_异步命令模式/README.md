# 场景2：异步命令模式

## 场景说明

演示如何使用异步命令模式。客户端发送命令后不等待响应，可以继续执行其他操作，稍后再获取响应结果。

## 运行方式

### 方式1：分别运行服务端和客户端

1. 启动服务端：
```bash
python 场景2_异步命令模式/service.py
```

2. 在另一个终端启动客户端：
```bash
python 场景2_异步命令模式/client.py
```

### 方式2：运行完整示例

```bash
python 场景2_异步命令模式/完整示例.py
```

## 功能说明

- **服务端**：提供数据处理服务，模拟耗时操作（处理时间 2 秒）
- **客户端**：发送异步命令，继续执行其他操作，然后获取结果

## 预期输出

服务端输出：
```
服务端已启动，等待命令...
收到命令: process_data, payload: {'data': 'task1'}
正在处理数据: task1 (耗时 2 秒)
处理完成: task1
```

客户端输出：
```
发送异步命令 1: task1
发送异步命令 2: task2
发送异步命令 3: task3
执行其他操作...
获取命令 1 的响应: {'result': 'task1_processed'}
获取命令 2 的响应: {'result': 'task2_processed'}
获取命令 3 的响应: {'result': 'task3_processed'}
```
