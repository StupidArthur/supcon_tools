# 场景4：服务注册与发现

## 场景说明

演示如何使用服务注册与发现功能。服务启动时自动注册，其他服务可以发现并调用已注册的服务。

## 运行方式

### 方式1：分别运行服务端和客户端

1. 启动服务端：
```bash
python 场景4_服务注册与发现/service.py
```

2. 在另一个终端启动客户端：
```bash
python 场景4_服务注册与发现/client.py
```

### 方式2：运行完整示例

```bash
python 场景4_服务注册与发现/完整示例.py
```

## 功能说明

- **服务端**：启动时注册服务，提供数据处理功能
- **客户端**：发现服务，调用服务，监控服务健康状态

## 预期输出

服务端输出：
```
服务已注册: data_service
服务端已启动，等待命令...
```

客户端输出：
```
发现服务: data_service
服务信息: {'name': 'data_service', 'metadata': {'version': '1.0.0'}, ...}
服务健康状态: True
调用服务: process, 参数: {'data': 'test_data'}
收到响应: {'result': 'test_data_processed'}
```
