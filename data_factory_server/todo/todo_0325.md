# 参数改动

前端在进行数据模拟、绘图时，每个program实例的全部参数都显示和绘图了，效果很不好，比如水箱的静态参数，PID的调参参数等等。
修改思路：
1. 每个program加一个属性 display_args，里面存放的是需要展示、动态运行的参数集，默认只有这些参数集才会被显示在数据模拟的前端，才会在数据导出的时候被导出（但是可在前端配置为当次模拟和导出增删参数）
2. 前端模拟配置的地方，增加一个位号配置的按钮，点击后弹出一个对话框，里面是所有位号及其参数的table，4列，位号名，参数名，参数描述，checkbox。第四列display_args默认勾选，人工可配置。配置完后，模拟出的绘图参数和导出参数，按配置的来。
这个改动不影响运行态
对于前端的开发，一定要控件化和结构化，不要平铺。
