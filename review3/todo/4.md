3. 组分编号与单位：不得改变
```text
component 1 = water
component 2 = ethanol
```
内部单位固定为：
量	单位
温度 `T`	K
压力 `P`、饱和蒸气压 `Psat`	kPa(a)
分子量	kg/kmol
液相、气相组成	mole fraction，0～1
NRTL `tau`、`gamma`	无量纲
NRTL 参数 `b12`、`b21`	K
禁止在热力学核心中混入 ℃、bar、mmHg 或质量分数。只有 Antoine 函数内部按公式要求把 K 转为 ℃。
4. Antoine 方程定稿
正式方程为：
[
\log_{10}\left(P_i^{sat}/\mathrm{kPa}\right)
= A_i-\frac{B_i}{T_{^\circ C}+C_i}
]
其中：
[
T_{^\circ C}=T_K-273.15
]
固定系数：
组分	A	B	C	输出压力
water	7.23255	1750.286	235.000	kPa
ethanol	7.16879	1552.601	222.419	kPa
建议常量定义：
```python
ANTOINE_WATER_KPA_C = (7.23255, 1750.286, 235.000)
ANTOINE_ETHANOL_KPA_C = (7.16879, 1552.601, 222.419)
```
实现必须明确写出单位，不允许只保留无语义的三元数组。
5. NRTL 方程和参数方向定稿
温度相关参数固定为：
[
\tau_{12}=3.46-\frac{586}{T_K}
]
[
\tau_{21}=-0.80+\frac{246}{T_K}
]
[
\alpha_{12}=\alpha_{21}=0.3
]
方向定义：
```text
tau_12 = tau_water_ethanol
tau_21 = tau_ethanol_water
```
`586` 和 `246` 的单位是 K；`tau` 为无量纲。建议不要使用容易互换的 `A12/A21` 裸名称，而使用完整常量名：
```python
NRTL_WATER_ETHANOL_A = 3.46
NRTL_WATER_ETHANOL_B_K = -586.0
NRTL_ETHANOL_WATER_A = -0.80
NRTL_ETHANOL_WATER_B_K = 246.0
NRTL_ALPHA = 0.3
```
因此代码表达式应为：
```python
tau_water_ethanol = NRTL_WATER_ETHANOL_A + NRTL_WATER_ETHANOL_B_K / temperature_k
tau_ethanol_water = NRTL_ETHANOL_WATER_A + NRTL_ETHANOL_WATER_B_K / temperature_k
```
在 `T=298.15 K` 时，必须得到：
```text
tau_water_ethanol  = 1.4945463692772092
tau_ethanol_water = 0.0250880429314104
```
定义：
[
G_{12}=\exp(-\alpha\tau_{12}),\qquad
G_{21}=\exp(-\alpha\tau_{21})
]
令：
```text
x1 = x_water = 1 - x_ethanol
x2 = x_ethanol
```
二元 NRTL 活度系数必须按以下闭式实现：
[
\ln\gamma_1=x_2^2\left[
\tau_{21}\left(\frac{G_{21}}{x_1+x_2G_{21}}\right)^2+
\frac{\tau_{12}G_{12}}{(x_2+x_1G_{12})^2}
\right]
]
[
\ln\gamma_2=x_1^2\left[
\tau_{12}\left(\frac{G_{12}}{x_2+x_1G_{12}}\right)^2+
\frac{\tau_{21}G_{21}}{(x_1+x_2G_{21})^2}
\right]
]
返回顺序固定为：
```python
gamma_water, gamma_ethanol
```
不要从通用多组分 NRTL 公式临时推导另一套下标约定。第一版二元后端直接采用以上闭式，降低参数方向出错概率。
6. 泡点和气相组成
第一版采用低压修正 Raoult 定律：
[
y_iP=x_i\gamma_iP_i^{sat}(T)
]
忽略气相逸度系数和 Poynting 修正。泡点残差为：
[
f(T)=x_1\gamma_1P_1^{sat}(T)+x_2\gamma_2P_2^{sat}(T)-P
]
找到 `f(T)=0` 后：
[
y_{ethanol}=\frac{x_{ethanol}\gamma_{ethanol}P_{ethanol}^{sat}}{P}
]
并令 `y_water = 1 - y_ethanol`。不要分别计算后再无条件归一化，以免归一化掩盖泡点求解残差。
正式接口：
```python
def saturation_pressure_kpa(component: str, temperature_k: float) -> float:
    ...

def nrtl_activity_coefficients(
    x_ethanol: float,
    temperature_k: float,
) -> tuple[float, float]:
    """Return (gamma_water, gamma_ethanol)."""
    ...

def bubble_point_temperature(
    x_ethanol: float,
    pressure_kpa: float,
    previous_temperature_k: float | None = None,
) -> tuple[float, float]:
    """Return (bubble_temperature_k, y_ethanol)."""
    ...
```
求根约束：
不增加 SciPy 依赖；
使用确定性的有界二分或自实现的有界混合求根；
对第一版支持的 `80～130 kPa(a)`，全局安全区间可用 `330～400 K`；
求根前验证两端残差异号，否则抛出包含 `x_ethanol`、`pressure_kpa` 和区间端点残差的异常；
最大迭代次数固定为 80；
温度区间容差 `1e-8 K`，或泡点压力残差 `1e-9 kPa`；
`previous_temperature_k` 只能用于缩小有效括区或加速，不能改变最终解；
不收敛时抛出明确异常，禁止返回 `NaN` 或沿用上周期值；
入口组成只接受 `0 <= x_ethanol <= 1`，压力必须有限且大于 0；无效输入直接 `ValueError`，不要静默钳位。
端点 `x_ethanol=0` 和 `1` 可以走纯组分快速路径，但结果必须与通用求根一致。
