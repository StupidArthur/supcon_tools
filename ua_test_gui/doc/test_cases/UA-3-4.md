## 执行与清理

1. 历史记录不假设可逐条清理；使用唯一位号、值域和时间窗隔离；
2. 删除临时位号配置；
3. 保存夹具清单、预期点集合和实际全集；
4. 性能大数据使用专用位号，避免污染功能夹具。
