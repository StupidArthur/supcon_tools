# UA-2-3 位号导入导出

验证用户能否将已有位号导出为可复用文件，通过 Excel 批量创建或更新位号，并从数据源选择底层节点批量注册位号。

本文件按用户任务组织，不按 API 拆分。API 仅用于构造数据、执行等价操作或独立核验。

---

## 设计约定

### 功能边界

本二级点覆盖：

1. 选择位号并导出 Excel；
2. 校验导出文件结构和内容；
3. 使用 Excel 批量创建或更新位号；
4. 处理跳过、覆盖和错误文件；
5. 完成“导出—离线修改—重新导入”闭环；
6. 从数据源浏览并批量注册底层节点。

不覆盖历史值 JSON、Excel/ZIP 或 CSV 导入；这些属于数据采集和历史数据能力。

### 与其他文件的边界

- `UA-2-1` 负责单个位号新增及一个批量新增冒烟闭环；完整批量导入场景由本文件负责。
- `UA-2-2` 负责查找已注册位号和浏览
