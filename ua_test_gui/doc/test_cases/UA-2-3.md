# UA-2-3 位号导入导出

验证通过现有 API 能否导出已有位号、使用 Excel 批量创建或更新位号，并从数据源批量注册底层节点。

本版本仅设计可由导入导出 API、查询 API、RT 和 asyncua 执行并验证的用例。前端文件选择器、按钮状态、跨页选择和交互提示不进入当前回归范围。

---

## 设计约定

- 用例按导出、Excel 导入、往返闭环和数据源批量注册等用户任务组织。
- 导入成功必须回查数量、配置和可用性，不能只检查 `code=00000`。
- 导出结果按输入 ID 集合与文件内容核对，不依赖页面选择状态。
- 可注册节点集合使用 `dsId + tagBaseName` 与已注册集合做差。
- `GUI-DEFERRED` 只保留原编号占位，不设计步骤。
- 临时数据使用 `ua23_{runId
