# UA-3-3 实时写入

验证 `writeTagValues` 的单个、批量、类型、失败隔离和写后闭环。需要历史数据时按[历史数据制造指南](../history-data-fixtures.md)执行。

## 约定

- 返回 `tagNames` 为成功项，`failMsg` 为失败项。
- 写入前保存源端值、RT、质量；结束后恢复。
- 是否真正回写 OPC UA 源端尚未确认，相关用例为探索。
- 临时位号使用 `ua33_{runId}_` 前缀。

## 执行与清理

1. 恢复源端和实时值；
2. 删除临时位号；
3. 历史造数失败标记 `setup_failed`；
4. 保存成功名单、失败映射和读回证据。
