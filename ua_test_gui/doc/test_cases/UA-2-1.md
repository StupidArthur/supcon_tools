# UA-2-1 位号新增

注册一个或批量：按位号类型分类（一次/二次/虚）、数据源状态 4 种、底层位号 tagBaseName 规范与跨数据源、系统位号名 tagName 全局唯一、13 种数据类型的写入/读取/边界、单位、只读 2×2、采集频率、量程上下限超限处理、高/低限三档一致性、实时推送、位号描述、添加成功后可用性。
