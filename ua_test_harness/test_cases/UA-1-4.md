# UA-1-4 多数据源并存

多数据源隔离性：同时采集、一源断连/启停不影响其他、跨数据源相同底层位号、系统位号名全局唯一。

| 编号 | 三级点 | 前置条件 | 测试步骤 | 预期结果 | 验证手段 |
|------|--------|---------|---------|---------|---------|
| UA-1-4-01 | 两数据源各自正常采集 | mock-A(18950)、mock-B(18951) 均已启动，配置 change=true | 1. 分别注册 ds-A、ds-B 2. 各注册位号 3. 等待采集 4. 各自 getRTValue 2 次确认值在变化 | 两个 ds 均 alive=true；各自位号 RT 值 = 各自 mock 节点当前值 | list_ds_info + getRTValue |
| UA-1-4-02 | 一源断连不影响另一源 | UA-1-4-01 状态 | 1. 停止 mock-A 2. 等待 2s 3. 验证 ds-A 和 ds-B | ds-A alive=false、quality 降级；ds-B 不受影响、alive=true、值继续变化 | list_ds_info + getRTValue |
| UA-1-4-03 | 一源恢复不影响另一源 | UA-1-4-02 后 | 1. 重启 mock-A 2. 等待重连 3. 验证 ds-A 和 ds-B | ds-A 恢复 alive=true、采集恢复；ds-B 全程不受影响 | list_ds_info + getRTValue |
